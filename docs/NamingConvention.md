# Naming convention

| Item          | Convention            | Notes                         |
|---------------|-----------------------|-------------------------------|
| Class         | `PascalCase`          |                               |
| Struct        | `PascalCase`          |                               |
| Enum          | `PascalCase`          |                               |
| Enum value    | `PascalCase`          |                               |
| Function      | `PascalCase`          |                               |
| Method        | `PascalCase`          |                               |
| Variable      | `camelCase`           |                               |
| Constant      | `UPPER_SNAKE_CASE`    |                               |
| Field         | `_camelCase`          | Use `_` as field name prefix  |
| Public field  | `PascalCase`          |                               |
| Property      | `PascalCase`          |                               |
